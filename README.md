# React Demos — (Concepts Explained in Reels)

Repo contains a growing collection of **short from, React and Next.js demos** designed for fast learning and visual understanding.  
Each demo corresponds to a **1‑minute educational video** that breaks down a single concept with clear examples, animations, and practical use cases.

This repository is part of my ongoing series where I teach modern frontend concepts through **micro‑tutorials** — React hooks, state patterns, data fetching, caching, arrays, SQL integration, and more.

---

## 🎯 What You’ll Find Here

Turotials can be found in _/app/prev-classes_ organised by accurate naming contains:

- A **focused code demo** (React or Next.js)
- A **visual explanation** of how the concept behaves
- A link to the **video tutorial**
- Notes on when and why to use the pattern

### Current & Upcoming Concepts

- `useContext` — sharing state across components
- `useEffect` — mount behavior, dependency arrays, re-renders
- `react-query` — fetching, caching, stale‑while‑revalidate
- `SQL: Array[] vs Array()` — syntax differences and shortcomings
- `Next.js routing` — pages, layouts, dynamic segments
- `SQL basics` — connecting frontend demos to real data
- More concepts added weekly
